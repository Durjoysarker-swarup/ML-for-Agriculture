# Customer Segmentation

## 📋 Quick Overview

**Problem:** Segment customers into groups based on spending patterns

**Approach:** Unsupervised learning using K-Means clustering

**Skills Demonstrated:**
- Unsupervised learning fundamentals
- K-Means clustering algorithm
- Determining optimal number of clusters (Elbow method)
- Cluster profiling and interpretation
- Multi-dimensional data visualization

**Dataset:** Wholesale Customers Data (440 records, 8 features)

---

## 🎯 Why This Project Matters

**Business Context:**
Customer segmentation enables targeted strategies for different customer groups. Unlike supervised learning, we discover patterns without predefined labels.

**Agricultural Connection:**
- Farmer typology: Smallholder vs. commercial farmers
- Market segmentation: Different crops for different regions
- Regional clustering: Identify zones with similar characteristics
- Resource allocation: Target interventions by farmer segment
- Spatial clustering: Group fields by soil/climate similarity
