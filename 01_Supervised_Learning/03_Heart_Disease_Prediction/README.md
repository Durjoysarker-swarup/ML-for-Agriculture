# Heart Disease Prediction

## 📋 Quick Overview

**Problem:** Predict presence of heart disease from patient clinical measurements

**Approach:** Compare multiple classification algorithms

**Skills Demonstrated:**
- Algorithm comparison (Logistic Regression vs. Random Forest)
- Feature importance analysis
- Multiple evaluation metrics
- Ensemble methods basics
- Model interpretation

**Dataset:** Heart Disease dataset (303 patients, 13 clinical features)

---

## 🎯 Why This Project Matters

**Medical Context:**
Comparison of simple vs. complex models reveals how different algorithms capture different patterns in the data.

**Agricultural Connection:**
- Comparing simple heuristics vs. ML models for crop decisions
- Understanding which factors most influence yield
- Ensemble methods for combining predictions from multiple sensors
- Interpretability vs. accuracy tradeoff
