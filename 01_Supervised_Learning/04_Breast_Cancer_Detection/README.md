# Breast Cancer Detection

## 📋 Quick Overview

**Problem:** Classify tumors as malignant or benign from medical measurements

**Approach:** Support Vector Machines (SVM) for precise classification

**Skills Demonstrated:**
- Support Vector Machines basics
- Kernel selection and hyperparameter tuning
- Medical image feature analysis
- ROC-AUC curve interpretation
- Model evaluation for high-stakes problems

**Dataset:** Wisconsin Breast Cancer Dataset (569 samples, 30 features from cell nuclei)

---

## 🎯 Why This Project Matters

**Medical Context:**
SVMs are particularly effective for binary classification in high-dimensional spaces. This project introduces kernel methods, important for geospatial ML.

**Agricultural Connection:**
- Using computer vision features for crop/plant classification
- SVM kernels for non-linear decision boundaries
- High-stakes classification (disease vs. healthy)
- Image feature extraction from medical → agricultural imagery
