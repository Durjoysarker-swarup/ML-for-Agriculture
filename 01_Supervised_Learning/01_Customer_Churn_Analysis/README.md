# Customer Churn Analysis

## 📋 Quick Overview

**Problem:** Predict which customers are likely to leave a service

**Approach:** Binary classification using logistic regression

**Skills Demonstrated:**
- Exploratory Data Analysis (EDA)
- Data visualization
- Feature scaling and normalization
- Logistic regression
- Model interpretation

**Dataset:** Telecom customer data (10,000 records, 21 features)

---

## 🎯 Why This Project Matters

**General Context:**
Understanding churn patterns is critical for any service/business. This project teaches the foundational workflow for binary classification problems.

**Agricultural Connection:**
- Predicting farmer retention in agricultural programs
- Understanding which extension services are effective
- Identifying at-risk farmers who might abandon climate-smart practices
- Program effectiveness evaluation
